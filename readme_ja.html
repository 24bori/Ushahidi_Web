システム要件

  1. Unicode supportのあるサーバ。
  2. MySQL バージョン 5.0 以上。
  3. PHP バージョン 5.2.3 以上。
  4. HTTP サーバ。Ushahidiが構築されているKohanaフレームワークは次のWebサーバでの動作が確認されています: Apache 1.3+, Apache2.0+, lighttpd, 及びMS IIS。

必須エクステンション

以下はUshahidiを正しく動作させるためにあなたのサーバにインストールされていなければならないPHPエクステンションのリストです。

  1. PCRE はUTF-8の機能を正しく動作させるためには ?enable-utf8 及び ?enable-unicode-properties でコンパイルされていなければなりません。
  2. iconv はUTF-8音訳に必須です。
  3. mcrypt は暗号化に必須です。
  4. SPL はいくつかのコアライブラリに必須です。
  5. mbstring はKohanaのUTF-8機能をスピードアップします。
  6. MySQL はデータベースアクセスに必要です

あなたのサーバにインストール済のエクステンションを確認したいですか? その手順はこちら。

セットアップ事前準備

Ushahiduインストールの前にあなたのサーバのファイルとフォルダにいくつか書き込み権限を与える必要があります。あなたのサーバで以下のファイルとフォルダが書き込み可能になっているか確認してください:

  • application/config/config.php
  • application/config
  • application/cache
  • application/logs
  • media/uploads
  • .htaccess

OSによりファイルパーミッションの構成手順は様々に異なります。こちらにいくつかのOS用のパーミッションについての役に立つリンクがあります。Windows 及びUnix オペレーティングシステム。

インストレーションガイド

Ushahidi は手動でもWebインストーラ経由でもインストールできます。Ushahidiをインストールする最も簡単な方法はWebインストーラを使う方法で、全てステップバイステップで案内してくれます。

Web インストーラ

  1. 解凍したUshahidi のファイル群をあなたのサーバにアップロード。
  2. あなたのWebサーバで適切なファイルが書き込み可能になっていることを確認(上記 "セットアップ事前準備" 参照)します。
  3. ブラウザにhttp://[Your Ushahidi URL]/installer を入力し、ステップバイステップの命令に従います。

手動インストレーション

  1. 解凍したUshahidi ファイル群をあなたのサーバにアップロード。
  2. データベースを作成し、接続できているかどうか注意してください。
  3. あなたのWebサーバで適切なファイルが書き込み可能になっていることを確認(上記 "セットアップ事前準備" 参照)。
  4. ./application/config/database.template.php を ./application/config/database.phpにコピーまたはリネーム。
  5. database.php を編集してデータベース接続情報を書き込んでください。
  6. ./application/config/config.template.php を ./application/config/config.phpにコピーまたはリネーム。
  7. config.php を編集して適切な値を設定してください。ファイル内のコメントを見れば正確な設定値が分かります。たいていの場合、その多くは変更不要です。
  8. 次のファイル内のSQL記述を実行してあなたのデータベース内に適切なテーブルを作成します。: ./sql/ushahidi.sql
  9. あなたのサイトをブラウズします:
    • URL: http://[Your Ushahidi URL]/
    • 管理エリア: http://[Your Ushahidi URL]/admin
    • ユーザ: admin
    • パスワード: admin

注意: あなたのインスタンスを完全に最適化するにはまだいくつかの設定と構成があります。これらの設定についての詳細は我々のwiki上にあります。
